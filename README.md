## 📦 주요 기능

### ⭐️ 나의 구직 관리

- 이력서 관리
- 자기소개서 관리
- 면접 일정 관리
- 회사별 메모 관리

### ⭐️ 채용 플랫폼 비교

- 주요 채용 플랫폼 정보 제공
- 각 채용 플랫폼의 특징과 장단점 비교 제공

### ⭐️ 달력으로 알기 쉬운 구직 관리

- 일간, 주간, 월간 일정 표시 기능 제공
- 면접, 채용 마감일, 자유 일정 메모 등 구직 관리에 필요한 내용을 달력에서 관리
